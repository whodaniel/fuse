The New Fuse with GitHub Copilot Guide

This guide will help you use The New Fuse extension with GitHub Copilot to create collaborative AI workflows.

  • Activating The New Fuse Extension
  • Creating a Collaborative Workflow
  • Example Workflow
  • Using JavaScript Scripts
  • Troubleshooting
Prerequisites: Both The New Fuse and GitHub Copilot extensions are already installed in your VS Code environment. The Redis server is running on port 6379.

Activating The New Fuse Extension

Step 1: Open VS Code Command Palette

Press Cmd+Shift+P (Mac) or Ctrl+Shift+P (Windows/Linux) to open the VS Code Command Palette.

Step 2: Show Discovered AI Agents

Type the following command in the Command Palette and select it:

The New Fuse: Show Discovered AI Agents

Step 3: Refresh Agent Discovery

Open the Command Palette again and type:

The New Fuse: Refresh Agent Discovery

Step 4: Open Workflow Builder

Open the Command Palette again and type:

The New Fuse: Open Workflow Builder

Creating a Collaborative Workflow

Step 1: Create a New Workflow

In the workflow builder, click the "New Workflow" button and provide a name and description for your workflow.

Step 2: Add a GitHub Copilot Step

Drag and drop a "Code Generation" step type from the sidebar onto the canvas. Configure it to use GitHub Copilot as the agent.

Step 3: Add a The New Fuse Step

Drag and drop a "Code Explanation" step type from the sidebar onto the canvas. Configure it to use The New Fuse as the agent.

Step 4: Connect the Steps

Click and drag from the output port of the Copilot step to the input port of The New Fuse step to create a connection.

Step 5: Save and Execute the Workflow

Click the "Save" button to save your workflow, then click "Execute" to run it.

Using JavaScript Scripts

Open Workflow Builder

Run the following script in a JavaScript Debug Terminal:

node open-workflow-builder.js

Show Discovered Agents

Run the following script in a JavaScript Debug Terminal:

node show-agents.js

Refresh Agent Discovery

Run the following script in a JavaScript Debug Terminal:

node refresh-agents.js

Create a Collaborative Workflow

Run the following script in a JavaScript Debug Terminal:

node create-collaborative-workflow.js
Note: These scripts require the VS Code API to be available. They should be run in a JavaScript Debug Terminal within VS Code.

Troubleshooting

Check Extension Status

Run the following command in a terminal to verify that both extensions are installed:

code --list-extensions | grep -E 'fuse|copilot'

Check Redis Server

Run the following command in a terminal to verify that the Redis server is running:

ps aux | grep redis

Restart VS Code

If you encounter issues, try restarting VS Code and then activating The New Fuse extension again.
